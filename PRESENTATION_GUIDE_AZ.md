# LogiConnect Satış Təqdimatı - İstifadə Təlimatı

## 📊 Təqdimat Səhifəsi Haqqında

LogiConnect platforması üçün xüsusi olaraq **satış təqdimatı səhifəsi** hazırlanmışdır. Bu səhifə LogiConnect əməkdaşlarının potensial müştərilərə (təchizatçı və logistika şirkətlərinə) platformanı təqdim etməsi və satışı həyata keçirməsi üçün nəzərdə tutulmuşdur.

## 🔗 Giriş Linki

Təqdimat səhifəsinə daxil olmaq üçün:
```
http://localhost:3000/az/presentation
```

Və ya canlı versiyada:
```
https://logiconnect.netlify.app/az/presentation
```

## 📋 Təqdimatın Strukturu

### 1. **Hero Bölməsi** - Güclü Açılış
- Platform logosu və brending
- Təsirli başlıq: "Logistikada Yeni Dövr"
- Əsas statistikalar:
  - 20-40% qənaət
  - 100+ logistika partnyor
  - 4 dil dəstəyi
  - 24/7 dəstək
- İki CTA düyməsi: Təchizatçı və Logistika qeydiyyatı

### 2. **Problem Bölməsi** - Bazardakı Problemlər
Müştərilərə mövcud problemləri göstərir:

**Təchizatçıların problemləri:**
- Yüksək qiymətlər və qeyri-şəffaf tariflər
- Təkliflərin müqayisəsində vaxt itkisi
- Etibarlı logistika şirkətlərinin tapılması çətinliyi
- Yükün izlənməsində problemlər

**Logistika şirkətlərinin problemləri:**
- Yeni müştəri tapmaq üçün böyük satış xərcləri
- Boş daşıma gücü
- Bazar məlumatı çatışmazlığı
- Müştəri idarəetməsi problemləri

### 3. **Həll Bölməsi** - LogiConnect Platforması
- 4 nəqliyyat növü göstərilir: Hava, Dəniz, Dəmiryolu, Avtomobil
- Platformanın əsas funksiyaları

### 4. **Təchizatçılar Üçün** - Ətraflı Xüsusiyyətlər
6 əsas xüsusiyyət:
1. **20-40% Qənaət** - Rəqabətli təkliflər
2. **Sadələşdirilmiş Proses** - Sürətli elan yaratma
3. **Tam Şəffaflıq** - Təkliflərin müqayisəsi
4. **Ətraflı Analitika** - Dashboard və hesabatlar
5. **Komanda İşi** - Çoxsaylı işçi dəstəyi
6. **Real-vaxt İzləmə** - Göndərişlərin izlənməsi

### 5. **Logistika Şirkətləri Üçün** - Ətraflı Xüsusiyyətlər
6 əsas xüsusiyyət:
1. **Avtomatik Müştəri Əldə Etmə** - 100+ gündəlik imkan
2. **Çevik Təklif Sistemi** - Çoxlu nəqliyyat növləri
3. **Gəlir Artımı** - Rəqabətli qiymətlərlə satış
4. **Rəqabət Analizi** - Qiymət və uğur dərəcəsi
5. **Əməliyyat Səmərəliliyi** - Status yeniləmələri
6. **Ətraflı Hesabatlar** - CSV ixracı və analitika

### 6. **Necə İşləyir?** - 3 Addım
1. **Elan Yaradın** - Təchizatçılar yük detallarını daxil edir
2. **Təkliflər Alın** - Logistika şirkətləri təklif göndərir
3. **Ən Yaxşısını Seçin** - Təchizatçı uyğun təklifi seçir

### 7. **Əsas Üstünlüklər**
- Təhlükəsizlik və Etibarlılıq
- Qlobal Hazır (4 dil)
- Mobil Optimizasiya
- Korporativ Səviyyə

### 8. **Texnologiya Stack**
- Next.js 14
- PostgreSQL
- TypeScript
- Tailwind CSS

### 9. **Final CTA** - Güclü Çağırış
- Böyük, rəngli çağırış bloqu
- İki qeydiyyat seçimi
- Pulsuz və sürətli qeydiyyat qeydi
